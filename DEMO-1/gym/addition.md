*_init_.py
### 总结
1. **标识包**：告诉 Python 这是一个可导入的包
2. **简化导入**：让外部不用写长长的路径
3. **控制导出**：规定 `import *` 能导入什么
4. **初始化**：包被导入时自动执行代码

✅ **os**：管 **文件、文件夹、路径、操作系统**
✅ **sys**：管 **Python 自己、导入路径、命令行、程序退出**


|代码|意思|
|---|---|
|`obj.key`|获取 obj 里**固定名字**叫 key 的属性|
|`getattr(obj, key)`|获取 obj 里**变量 key 代表的名字**的属性|
