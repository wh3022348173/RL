### 1. 射线属性定义

- **单色 / 多色光**：默认单色，可指定离散波长 / 频率列表，模拟复色光COMSOL。
- **偏振**：内置 Stokes-Mueller 矩阵计算，支持**完全偏振、非偏振、部分偏振**射线追踪COMSOL中国。
- **强度 / 功率**：可追踪射线强度、光功率，考虑介质吸收、界面菲涅尔损耗。
- **介质类型**：支持**均匀折射率介质**、**渐变折射率（GRIN）介质**（折射率随空间坐标变化）COMSOL中国。

### 2. 射线释放（Ray Release）：光源定义

提供多种射线释放特征，模拟不同类型光源COMSOL：

1. **网格释放（Grid）**：在平面 / 曲面上按网格均匀释放射线，模拟平行光、面光源。
2. **点释放（Point）**：从点源向指定方向 / 立体角释放射线，模拟点光源、激光。
3. **高斯光束（Gaussian Beam）**：释放强度呈高斯分布的射线，可定义束腰、发散角、瑞利范围。
4. **边界释放（Boundary）**：从模型边界释放射线，模拟反射 / 折射后的二次光源。
5. **太阳辐射（Solar Radiation）**：预设太阳光谱、入射方向，用于太阳能光学系统。

### 3. 边界条件（Boundary Conditions）：界面行为

定义射线遇到介质分界面时的行为，支持组合设置COMSOL：

- **反射 / 折射（默认）**：光滑界面自动计算反射、折射射线方向与强度（菲涅尔公式）。
- **镜面反射（Specular）**：理想镜面反射，无漫反射。
- **漫反射（Diffuse）**：射线按朗伯分布散射，模拟粗糙表面。
- **吸收（Absorption）**：射线被界面吸收，可选择保留 / 删除射线。
- **光栅衍射（Diffraction Grating）**：自动计算多级衍射（反射 / 透射）的方向与效率COMSOL。
- **光阑（Aperture）**：阻挡 / 透射射线，模拟孔径、遮光片。

### 4. 域条件（Domain Conditions）：介质内行为

- **吸收介质**：定义衰减系数、内部透射率，模拟材料对光的吸收（如有色玻璃、吸收涂层）。
- **渐变折射率（GRIN）**：通过表达式 / 函数定义折射率分布，模拟自聚焦透镜、梯度光纤。
- **射线加热（Ray Heating）**：射线能量被介质吸收转化为热量，实现**射线光学→传热**单向 / 双向耦合（热透镜效应）COMSOL。
