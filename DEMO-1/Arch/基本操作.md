## 系统基础 & 终端常用命令
```bash
# 目录/文件操作 (最常用) 
ls # 查看当前目录文件/文件夹 
ls -a # 显示所有文件(含隐藏文件 .xxx) 
ls -l # 详细信息(权限/大小/时间)，简写 ll 
cd 目录名 # 进入指定目录，
cd .. # 返回上一级目录 
cd ~ # 回到用户主目录 /home/lolita 
cd - # 回到上一次所在目录 

# 文件操作 
pwd # 查看当前所在的绝对路径 (必备) 
mkdir 文件夹名 # 创建文件夹，
mkdir -p a/b/c # 递归创建多级文件夹 (必备) 
rm 文件 # 删除文件 
rm -rf 文件夹 # 强制删除文件夹/文件，无提示
cp 源文件 目标路径 # 复制文件，
mv 源文件 目标路径 # 移动/重命名文件，
touch 文件名 # 创建空白文件

# 权限管理 (必备，配置脚本/程序必用) 
chmod +x 文件名 # 赋予文件【执行权限】
sudo chmod 777 文件 # 赋予所有用户读写执行权限
```
## pacman
```bash
# 基础核心三件套  
sudo pacman -S 软件名 # 安装软件，
sudo pacman -Ss 软件名 # 搜索软件包，
sudo pacman -R 软件名 # 卸载软件，保留配置文件 
sudo pacman -Rns 软件名 # 彻底卸载软件+删除配置文件+依赖 (

# 系统更新/维护  
sudo pacman -Syyu # 全系统更新，同步源+升级所有包 (最常用，安全稳定) 
sudo pacman -Sy # 只同步软件源数据库，不升级 
sudo pacman -Su # 只升级已安装的软件包 
# 清理缓存 (释放磁盘空间，必备) 
sudo pacman -Sc # 清理无用的旧缓存 (安全) 
sudo pacman -Scc # 清理所有缓存 (谨慎，重装软件需要重新下载) 

# 其他实用 
pacman -Qi 软件名 # 查看已安装软件的详细信息 
pacman -Ql 软件名 # 查看软件安装的所有文件路径 
sudo pacman -U 本地包名 # 安装本地下载的 .pkg.tar.zst 包
```

## yay
```bash
# 基础核心  
yay -S 软件名 # 安装AUR软件，
yay -Ss 软件名 # 搜索AUR+官方源的软件 
yay -Rns 软件名 # 彻底卸载AUR软件 
yay -Syu # 更新【所有软件】：官方源+AUR源 (推荐，一键全更) 

# 清理缓存  
yay -Sc # 清理无用缓存 
rm -rf ~/.cache/yay # 彻底清理yay所有缓存 
```
## 权限
```bash
sudo 命令 # 以管理员(root)权限执行命令
su # 切换到root用户 
exit # 退出root用户/关闭终端会话 
whoami # 查看当前登录用户名 
passwd # 修改当前用户密码 
sudo passwd root # 修改root用户密码
```

## 进程 / 服务管理命令
### 进程管理
```bash
ps -aux | grep 程序名 # 查看程序是否运行
ps -ef # 查看系统所有运行的进程 
sudo pkill -9 程序名 # 强制结束进程，
top # 实时查看系统进程/CPU/内存占用 (终端任务管理器) 
htop # top的美化版，需要安装: sudo pacman -S htop (推荐)
```
### systemd 系统服务管理
```bash
sudo systemctl status 服务名 # 查看服务运行状态
sudo systemctl start 服务名 # 启动服务 
sudo systemctl stop 服务名 # 停止服务 
sudo systemctl restart 服务名 # 重启服务 
sudo systemctl enable 服务名 # 设置服务开机自启 
sudo systemctl disable 服务名 # 关闭服务开机自启 
sudo systemctl enable --now 服务名 # 开机自启 + 立即启动 (一步到位，必备)
```
## 网络相关命令
```bash
ping 域名/IP # 测试网络连通性
curl -I 域名 # 测试网站访问状态
curl -x http://127.0.0.1:38457 -I github.com # 测试FastGithub代理是否生效 (你的专属) 
ip addr # 查看本机所有网卡/IP地址 (简写 ip a) 
nmcli device status # 查看网络连接状态 (WiFi/有线) 
sudo nmcli networking off/on # 关闭/开启网络
```
