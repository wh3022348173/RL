## 选题意义

### 课题来源

本课题源于工业生产安全监测与环境空气质量检测的实际需求。随着工业智能化进程的加速和环境保护意识的提升，对气体浓度的实时、精准检测需求日益迫切。非分散红外（NDIR）技术凭借选择性强、稳定性高、抗干扰能力突出等优势，在气体检测领域得到广泛关注，但现有基于 NDIR 的气体传感器存在检测精度优化不足、数据处理实时性欠缺、可视化呈现单一等问题。结合工业场景对传感器小型化、智能化的需求，本课题聚焦 NDIR 气体传感器的硬件设计与数据可视化系统开发，旨在提升气体检测的精准度与数据呈现的直观性。

### 理论研究价值

从技术角度来看，本课题的研究具有重要理论价值。当前 NDIR 气体传感器的核心技术集中在光源驱动、光学结构优化、信号调理等方面，现有研究在多气体检测交叉干扰抑制、微弱信号提取精度等方面仍有提升空间。本课题将深入研究 NDIR 检测的光学原理与信号处理算法，探索基于窄带滤光片的气体选择性增强技术和基于数字锁相放大的微弱信号提取方法，丰富 NDIR 气体传感技术的理论体系。同时，将传感器硬件设计与数据可视化技术深度融合，为智能传感系统的一体化设计提供理论参考，对推动传感技术与数据处理技术的交叉融合具有积极意义。

### 实际应用价值

在经济与社会效益方面，本课题的研究成果具有显著实用价值。工业生产中，有毒有害气体泄漏可能引发安全事故，造成巨大经济损失和人员伤亡，基于本课题设计的高精度 NDIR 气体传感器，可实现对甲烷、一氧化碳、二氧化碳等多种气体的实时监测，为工业安全预警提供技术支撑，降低事故发生概率。在环境监测领域，该系统可精准检测大气中污染物浓度，为环境治理决策提供数据支持，助力生态环境保护。此外，系统采用的小型化硬件设计与便捷化可视化界面，可降低检测设备的生产成本与使用门槛，适用于工业车间、城市环境、智能家居等多种场景，具有广阔的市场应用前景，能够产生良好的经济效益与社会效益。

### 问题提出与主题明确

相关研究表明，现有 NDIR 气体传感器在复杂环境下的检测误差较高，平均误差可达 5%-8%，且数据处理多采用离线分析方式，无法满足实时监测需求（李刚，王健，刘广玉。基于 NDIR 技术的多气体传感器研究进展 [J]. 传感技术学报，2022, 35 (4):456-463.）。目前的解决方案主要包括优化光学结构、采用传统信号滤波算法等，但在多干扰因素共存场景下的精度提升效果有限，且缺乏一体化的数据可视化平台，导致检测数据的可读性与实用性不足。本课题聚焦 “基于 NDIR 的气体传感器设计及数据可视化系统开发”，核心研究内容包括：设计小型化、高精度的 NDIR 气体传感器硬件模块，开发基于嵌入式系统的实时数据处理算法，构建 Web 端与移动端协同的数据可视化平台，实现气体浓度的精准检测、实时传输与直观呈现。

## 国内外研究现状概述

### 国外研究现状

国外在 NDIR 气体传感技术领域的研究起步较早，技术相对成熟。美国 Honeywell 公司开发的 NDIR 气体传感器采用微机电系统（MEMS）工艺，实现了传感器的小型化与低功耗，检测响应时间小于 1 秒，但在低浓度气体检测场景下的稳定性有待提升（Honeywell International Inc. Miniature NDIR Gas Sensor Technical Datasheet [R]. 2023.）。德国 Sensirion 公司提出了基于双光束补偿技术的 NDIR 检测方案，有效抑制了光源漂移带来的误差，检测精度可达 ±2% FS，但该方案硬件结构复杂，成本较高（Schmid M, Ryser M, Kuttner R. A dual-beam NDIR sensor for high-precision CO₂ measurement [J]. IEEE Sensors Journal, 2021, 21 (12):13452-13459.）。在数据可视化方面，国外主流检测设备多采用专用软件进行数据呈现，如美国 Thermo Fisher Scientific 公司的气体检测系统，支持实时数据曲线显示与历史数据查询，但可视化界面的定制化程度较低，且缺乏跨平台访问功能。

### 国内研究现状

国内近年来在 NDIR 气体传感技术领域的研究进展迅速。哈尔滨工业大学团队设计了基于新型红外光源的 NDIR 传感器，通过优化光源驱动电路，提升了光源的稳定性与使用寿命，检测精度可达 ±3% FS，但在多气体同时检测方面尚未实现突破（张宏志，李艳，张伟。基于新型红外光源的 NDIR 气体传感器设计 [J]. 光学学报，2022, 42 (11):1128001-1128008.）。中科院半导体研究所提出了基于机器学习的 NDIR 信号处理算法，利用支持向量机（SVM）对检测信号进行校正，有效降低了交叉干扰带来的误差，但算法的实时性有待优化（王磊，陈亮，赵玲。基于 SVM 的 NDIR 多气体检测交叉干扰抑制算法 [J]. 仪器仪表学报，2023, 44 (3):201-209.）。在数据可视化方面，国内相关研究多聚焦于单一平台开发，如基于 LabVIEW 的桌面端可视化系统或基于 Android 的移动端应用，缺乏 Web 端与移动端的协同联动，数据共享与远程访问能力不足。

### 研究现状总结与不足

综合来看，国内外在 NDIR 气体传感器的硬件设计、信号处理算法等方面已开展大量研究，取得了一定成果，但仍存在以下不足：一是传感器在复杂环境下的检测精度与稳定性有待进一步提升，多气体检测的交叉干扰抑制技术不够成熟；二是数据处理算法的实时性不足，难以满足高动态范围的气体浓度检测需求；三是数据可视化系统缺乏跨平台协同能力，界面交互不够友好，数据呈现形式较为单一。本课题针对上述不足，将优化传感器光学结构与信号处理算法，开发跨平台的数据可视化系统，实现气体检测与数据呈现的一体化、智能化。

## 主要研究内容

### 1. NDIR 气体传感器硬件模块设计

- 光学结构设计：采用微型气室结构，选择合适的红外光源与窄带滤光片，优化光源与探测器的安装角度，减少光学损耗，提升气体吸收信号的强度；
- 电路模块设计：设计光源驱动电路，实现光源的恒流驱动与脉冲调制，保证光源输出的稳定性；开发信号调理电路，包括前置放大、滤波、锁相放大等模块，实现微弱红外信号的提取与放大；
- 嵌入式系统搭建：选用 STM32 单片机作为核心控制单元，集成 ADC 模块实现模拟信号的数字化转换，通过 I2C、SPI 等接口实现数据的传输与存储。

### 2. 数据处理算法开发

- 基线校正算法：采用滑动平均法与多项式拟合相结合的方式，消除光源漂移、环境温度变化等因素带来的基线漂移误差；
- 交叉干扰抑制算法：基于气体红外吸收光谱的特征，建立多气体吸收模型，利用偏最小二乘回归（PLSR）算法分离不同气体的吸收信号，抑制交叉干扰；
- 实时数据校准算法：结合标准气体标定实验，建立检测数据与实际气体浓度的映射关系，实现数据的实时校准，提升检测精度。

### 3. 数据可视化系统开发

- 数据传输模块：采用 Wi-Fi 与蓝牙双模通信方式，实现传感器与上位机（Web 端、移动端）的数据实时传输；
- Web 端可视化平台：基于 Vue.js 与 ECharts 开发，支持实时数据曲线显示、历史数据查询、数据统计分析、异常报警等功能；
- 移动端应用开发：基于 React Native 开发跨平台移动端应用，实现数据的实时查看、设备远程控制、报警信息推送等功能；
- 数据库设计：采用 MySQL 数据库存储检测数据、设备参数、用户信息等，保证数据的安全性与可扩展性。

### 4. 系统集成与性能测试

- 硬件与软件集成：将传感器硬件模块、数据处理算法与可视化系统进行集成，实现端到端的气体检测与数据呈现；
- 性能测试：在实验室环境下，对传感器的检测精度、响应时间、稳定性等指标进行测试；在模拟工业环境下，验证系统的抗干扰能力与实用性；
- 系统优化：根据测试结果，对硬件电路、算法参数与可视化界面进行优化，提升系统的整体性能。

## 拟采用的研究思路

### 研究方法

- 文献研究法：查阅 NDIR 气体传感技术、信号处理算法、数据可视化系统开发等相关领域的国内外文献，了解研究现状与发展趋势，为课题研究提供理论支撑；
- 实验法：设计传感器硬件实验平台，进行光学结构优化实验、电路性能测试实验、标准气体标定实验等，获取实验数据，验证硬件设计与算法的有效性；
- 仿真与编程法：利用 Multisim 软件对电路模块进行仿真分析，优化电路参数；采用 C 语言、Python、JavaScript 等编程语言，实现数据处理算法与可视化系统的开发。

### 技术路线

1. 前期准备阶段：调研国内外相关技术与产品，明确研究目标与技术指标，制定详细的研究方案；
2. 硬件设计阶段：完成光学结构设计、电路原理图绘制与 PCB 制作，搭建嵌入式系统硬件平台；
3. 算法开发阶段：基于硬件平台，开发基线校正、交叉干扰抑制、数据校准等算法，进行算法仿真与验证；
4. 可视化系统开发阶段：设计数据库结构，开发 Web 端与移动端可视化平台，实现数据传输与可视化呈现；
5. 系统集成与测试阶段：将硬件、算法与可视化系统集成，进行性能测试与优化，撰写研究报告与论文。

### 可行性论证

- 技术可行性：NDIR 气体传感技术、嵌入式系统开发、数据处理算法、可视化系统开发等均为成熟的技术体系，相关理论与实践案例丰富，且课题组具备相关的硬件设计、软件开发能力，能够保障课题的顺利开展；
- 设备与资源可行性：实验室具备红外光源、探测器、窄带滤光片、STM32 开发板、示波器、频谱分析仪等硬件设备，以及 Multisim、Keil、Vue.js、ECharts 等软件开发工具，能够满足实验与开发需求；
- 时间可行性：课题研究周期为 14 周，各阶段任务安排合理，能够在规定时间内完成传感器设计、算法开发、系统集成与测试等工作。
