下载安装
```bash
sudo apt update
sudo apt install 0ad
##
which 0ad
/usr/games/0ad
```
汉化
下载对应汉化包
https://releases.wildfiregames.com/locales/
```bash
mkdir -p ~/.local/share/0ad/mods
# 启动游戏发现usr目录
cp ~/下载/zh-lang-0.27.0.pyromod ~/.local/share/0ad/mods/
# 执行安装
pyrogenesis zh-lang-0.27.0.pyromod
```
enable mod即可
save and restart
