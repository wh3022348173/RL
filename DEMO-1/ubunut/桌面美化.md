任意浏览器在扩展功能搜索插件GNOME Shell，下载并启用，这里推荐火狐浏览器或Chrome浏览器。
打开扩展，或者直接进入https://extensions.gnome.org/  然后安装Blur my shell，user themes,bash to dock 三个组件，前者可以让你的任务栏，Topbar，和窗口变成半透明风格，后者用于更改图标和安装你喜欢的主题。
打开Blur my shell，调整设置和参数，让你需要的地方变为半透明。
## 拓展
```bash
# extension manager
 sudo apt install gnome-shell-extension-manager -y
# extension 
sudo apt install gnome-shell-extension-prefs
```
## 优化
```bash
sudo apt install gnome-tweaks 
```
