⭐ 一、【系统包管理 - 软件安装 / 卸载 / 更新】高频 TOP（重中之重）
Ubuntu 核心包管理工具是 apt，所有命令都需要 sudo 管理员权限，必记！
基础更新 / 升级
```bash
sudo apt update        # 更新系统软件包的「下载源列表」（必须先执行，更新索引）
sudo apt upgrade       # 升级所有已安装的软件包（保留配置，安全升级，推荐）
sudo apt full-upgrade  # 深度升级（会卸载旧依赖/冲突包，大版本更新用）
```
软件安装 / 卸载
```bash
sudo apt install 包名   # 安装指定软件包，例：sudo apt install nginx
sudo apt remove 包名    # 卸载软件「本体」，保留配置文件（后续重装可复用配置）
sudo apt purge 包名     # 彻底卸载「本体+所有配置文件」，干干净净，最推荐！
sudo apt autoremove     # 卸载软件后必执行！清理「无用的依赖残留包」，释放磁盘空间
```
软件包相关辅助
```bash
sudo apt clean         # 清空下载的软件包缓存（缓存目录/var/cache/apt/archives/）
sudo apt autoclean     # 只清理过期的缓存包（保留可用缓存，更温和）
apt search 关键词      # 搜索系统源里的软件包，例：apt search python3
apt show 包名          # 查看软件包的详细信息（版本、依赖、介绍）
apt list --installed   # 列出系统所有已安装的软件包
apt list --installed | grep 关键词  # 精准查找已安装的包，例：apt list --installed | grep docker
```
⭐ 二、【文件与目录操作】高频 TOP（每天必用，Linux 通用）
基础导航 / 查看（无权限限制，最常用）
```bash
pwd           # 查看「当前所在目录的完整路径」（必用，防止迷路）
ls            # 列出当前目录下的文件/文件夹（仅名称）
ls -l         # 详细列表：显示权限、所有者、大小、修改时间（简写 ll，效果一样）
ls -a         # 显示「所有文件」，包含隐藏文件（Linux中 .开头的是隐藏文件）
ls -lh        # 人性化显示文件大小（KB/MB/GB），代替字节，可读性拉满
cd 目录路径   # 切换目录，例：cd /home/ubuntu  cd ../ （返回上一级）
cd ~          # 一键回到「当前用户的家目录」，万能快捷
cd -          # 回到「上一次所在的目录」，来回切换超方便
```
文件 / 文件夹 增删改查（核心）
```bash
mkdir 文件夹名        # 创建空文件夹，例：mkdir test
mkdir -p a/b/c       # 递归创建多级文件夹（父目录不存在则自动创建）
touch 文件名         # 创建空文件，例：touch test.txt  touch index.py
cp 源文件 目标路径    # 复制文件，例：cp test.txt /home/ubuntu/
cp -r 源文件夹 目标路径 # 复制文件夹（必须加 -r 递归），例：cp -r test /home/
mv 源文件 目标路径    # 移动文件/文件夹（剪切），例：mv test.txt /tmp/
mv 旧文件名 新文件名  # 重命名文件/文件夹，例：mv test.txt new.txt
rm 文件名            # 删除文件（直接删除，无回收站，谨慎）
rm -r 文件夹名       # 删除文件夹（递归删除所有内容）
rm -rf 目标          # 强制删除「文件/文件夹」，无视权限和提示（慎用！删库跑路元凶）
```
文件查看 / 编辑（日常开发必用）
```bash
cat 文件名        # 一次性查看文件所有内容（适合小文件）
more 文件名       # 分页查看文件内容（按空格翻页，q退出，适合大文件）
less 文件名       # 增强版分页查看（支持上下翻页、搜索，q退出，大文件首选）
head 文件名       # 查看文件「前10行」
head -n 20 文件名 # 查看文件前20行，自定义行数
tail 文件名       # 查看文件「最后10行」
tail -f 文件名    # 实时监听文件新增内容（日志查看神器！例：tail -f /var/log/nginx/access.log）
```
⭐ 三、【文件权限管理】高频 TOP（解决 90% 的权限报错）
Linux 中「一切皆文件」，文件 / 文件夹的权限错误是最常见的问题，核心命令 chmod chown，需要 sudo 权限
权限修改（chmod，文件 / 文件夹通用）
```bash
sudo chmod 777 文件名/文件夹名  # 赋予「所有用户」读+写+执行权限（临时测试用，方便）
sudo chmod 755 文件名           # 最安全常用：所有者可读可写可执行，其他用户只读+执行
sudo chmod -R 755 文件夹名      # 递归修改文件夹内所有文件的权限（必须加 -R）
```
所有者修改（chown，解决「权限拒绝」核心）
```bash
sudo chown 用户名:用户组 文件名  # 修改文件的所有者，例：sudo chown ubuntu:ubuntu test.txt
sudo chown -R 用户名:用户组 文件夹名 # 递归修改文件夹所有者
sudo chown -R $USER:$USER 路径  # 万能写法：把文件所有者改为「当前登录用户」
```
⭐ 四、【用户与权限】基础命令（系统管理常用）
用户相关
```bash
sudo adduser 用户名    # 创建新用户（带家目录，自动配置权限，推荐）
sudo deluser 用户名    # 删除用户（保留家目录）
sudo deluser --remove-home 用户名 # 删除用户+删除其家目录（彻底删除）
sudo passwd 用户名     # 修改用户密码（包括root用户），输入后回车输新密码即可
su 用户名              # 切换到指定用户（需要输入该用户密码）
su - 用户名            # 切换用户+加载该用户的环境变量（推荐，切换后环境生效）
exit                  # 退出当前切换的用户，回到原用户
```
提权相关
```bash
sudo -i                # 切换到 root 超级管理员（拥有所有权限，谨慎使用）
exit                  # 从 root 用户退回普通用户
```

⭐ 六、【进程管理】高频 TOP（结束卡死程序 / 后台运行程序）
查看进程
```bash
ps -aux         # 列出系统「所有运行的进程」（详细信息：PID、用户、占用资源）
ps -aux | grep 关键词 # 精准查找进程，例：ps -aux | grep nginx 查找nginx进程
```
结束进程（解决程序卡死、强制关闭服务）
```bash
kill 进程PID     # 温和关闭进程（PID是进程编号，ps -aux查到的第二列）
kill -9 进程PID  # 强制杀死进程（最常用！进程卡死、无响应必用，强制终止）
pkill 进程名     # 按「进程名称」直接杀死进程，例：pkill nginx（无需记PID，更方便）
```

⭐ 八、【压缩与解压】高频 TOP（处理 zip/tar 包，日常必备）
Ubuntu 中最常见的压缩格式：tar.gz / tar.xz / zip，所有命令通用，记住就行！
tar 格式（Linux 最主流，重点记）
```bash
# 解压（万能解压命令，所有tar包通用）
tar -zxvf 压缩包名.tar.gz  # 解压 tar.gz 格式
tar -xvf 压缩包名.tar.xz   # 解压 tar.xz 格式
# 压缩
tar -zcvf 压缩后的包名.tar.gz 要压缩的文件/文件夹  # 例：tar -zcvf test.tar.gz test/
```
zip 格式（Windows 兼容，常用）
```bash
sudo apt install zip unzip  # 先安装解压工具（系统默认没有）
unzip 压缩包名.zip          # 解压zip包
zip -r 压缩后的包名.zip 要压缩的文件夹  # 压缩文件夹（必须加 -r）
```
⭐ 九、【实用快捷键 + 小命令】（效率翻倍，必记）
终端快捷键（每天省 10 分钟，Linux 终端通用）
```plaintext
Ctrl + C  → 终止当前正在运行的命令（程序卡死、运行错误，一键停止，最常用！）
Ctrl + L  → 清空终端屏幕（替代 clear 命令，更快）
Ctrl + D  → 退出终端/当前用户
Tab       → 自动补全「文件名/目录名/命令」（敲前2个字符按Tab，一键补全，神器）
↑ ↓       → 上下箭头，调用历史执行过的命令（不用重复敲）
```
其他实用小命令
```bash
clear       # 清空终端屏幕
history     # 查看所有历史执行过的命令
sudo reboot # 重启系统
sudo poweroff # 关机
man 命令名   # 查看命令的「官方详细手册」，例：man apt  man ls（不懂就查，万能帮助）
```
