## 软件包安装(.deb)
### 方法 1：dpkg 命令
```bash
# 核心安装命令 格式 
sudo dpkg -i 你的deb包完整路径/软件包名称.deb
```
#### dpkg 核心参数说明

- `-i` ：全称 `install`，核心功能 = 安装 / 配置 deb 包
- `-r` ：全称 `remove`，卸载软件（保留配置文件）
- `-P` ：全称 `purge`，彻底卸载软件（删除配置文件，推荐卸载用）
### 方法 2：apt install 本地安装
```bash
# 直接用 apt install 安装本地deb包，路径必须写完整 
sudo apt install ./你的软件包名称.deb
```
### 解决「安装 deb 包最常见的报错」：依赖缺失
#### 方案 1：执行系统提示的修复命令
```bash
sudo apt --fix-broken install
```
#### 方案 2：先修复再重新安装
```bash
# 1. 先修复系统依赖环境 
sudo apt update && sudo apt upgrade -y sudo apt --fix-broken install 
# 2. 重新安装deb包 
sudo dpkg -i 你的软件包名称.deb
```
